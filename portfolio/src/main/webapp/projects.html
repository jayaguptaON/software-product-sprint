Projects

SmartHire - March 2019

SmartHire is a hiring web-app that uses AI to eliminate bias in the hiring process. It analyzes hiring documents (eg. resumes, cover letters) to make recommendations about the top candidates to consider from a pool of applicants.

  • Built with Python, HTML, CSS, JavaScript
  • Used IBM's Natural Language Understanding APIs and open source model Word2Vec to evaluate the 'distance' (i.e. similarity) between candidates' qualifications and job requirements
  • Implemented post-processing method using IBM's AI Fairness 360 library to examine and mitigate bias in the model

See more: Devpost / GitHub

MeeLia - March 2019

MeeLia is a voice assistant geared towards helping young people, especially women, with mental health issues. It performs sentiment analysis of users' speech using machine learning to meaningfully converse with those experiencing mental health crisis.

  • Built with Python, HTML, CSS, and Bootstrap
  • Trained model using Google Cloud Natural Language API and DialogFlow
